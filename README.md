### This is my first sinatra app

A small app to manage your tasks

![demo](https://github.com/kesin/sinatra_task_manager/raw/master/public/task_manager.png)

### How to Run
	```shell
	clone this project
	cd sinatra_task_manager
	bundle install
	rake db:migrate
	ruby main.rb
	```
